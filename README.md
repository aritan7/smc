# TDS3751 SOCIAL MEDIA COMPUTING
Flask Application

### To install Flask
```python
pip install flask
```

### To start the web server
Mac / Linux:
```python
export FLASK_APP=application.py
flask run
```

Windows:
```python
set FLASK_APP=application.py
flask run
```

### Debug mode
Mac / Linux:
```python
export FLASK_DEBUG=1
```

Windows:
```python
set FLASK_DEBUG=1
```
